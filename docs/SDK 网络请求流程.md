## Flutter Ali HttpDNS SDK еҠҹиғҪеҲҶжһҗ

### 1. SDK ж ёеҝғеҠҹиғҪ

**дё»иҰҒз»„д»¶пјҡ**
- `FlutterAliHttpDns`: дё»жҸ’д»¶зұ»пјҲеҚ•дҫӢжЁЎејҸпјү
- `DnsResolver`: DNSи§ЈжһҗеҷЁпјҲжң¬ең°зј“еӯҳ + е№іеҸ°и§Јжһҗпјү
- `ProxyServer`: жҷәиғҪд»ЈзҗҶжңҚеҠЎеҷЁ
- `MappingUtils`: еҹҹеҗҚе’Ңз«ҜеҸЈжҳ е°„е·Ҙе…·
- `Http1Handler`: HTTP/1.1еҚҸи®®еӨ„зҗҶеҷЁ
- `Http2Handler`: HTTP/2еҚҸи®®еӨ„зҗҶеҷЁ

2. **еҗҜеҠЁд»ЈзҗҶ**: `startProxy()` - еҗҜеҠЁжң¬ең°д»ЈзҗҶжңҚеҠЎеҷЁ
3. **й…ҚзҪ®е®ўжҲ·з«Ҝ**: 
   - Dio: дҪҝз”Ё `getDioProxyConfig()` иҺ·еҸ–д»ЈзҗҶй…ҚзҪ®
   - HttpClient: дҪҝз”Ё `configureHttpClient()` й…ҚзҪ®д»ЈзҗҶ
4. **иҜ·жұӮеӨ„зҗҶ**:
   - е®ўжҲ·з«ҜеҸ‘йҖҒиҜ·жұӮеҲ° `localhost:port`
   - д»ЈзҗҶжңҚеҠЎеҷЁжҺҘж”¶иҜ·жұӮ
   - дҪҝз”Ё `MappingUtils` еә”з”ЁеҹҹеҗҚжҳ е°„
   - дҪҝз”Ё `DnsResolver` и§Јжһҗзӣ®ж ҮеҹҹеҗҚ
   - иҪ¬еҸ‘иҜ·жұӮеҲ°зӣ®ж ҮжңҚеҠЎеҷЁ
