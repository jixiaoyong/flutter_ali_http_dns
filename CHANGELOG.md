# Changelog

## [0.0.1] - 2025-08-22

### ✨ 新增功能
- 阿里云 HttpDNS SDK 集成
- 域名解析功能
- 智能 HTTP 代理服务器
- 支持 HttpClient 和 Dio 集成
- 端口映射和域名映射
- 缓存管理和 IPv6 支持
- 预加载域名和测速功能

### 🔧 技术特性
- `DnsConfig` - DNS 配置模型
- `ProxyConfig` - 代理配置模型  
- `FlutterAliHttpDns` - 主插件类
- 支持 Android (API 21+) 和 iOS (12.0+)
- 完整的错误处理和生命周期管理

### 📚 文档
- 完整的示例项目
- 详细的使用文档和 API 参考
- 常见问题解答
