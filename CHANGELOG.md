# Changelog

## [0.0.1] - 2025-08-28

### 🎉 初始版本发布

#### ✨ 核心功能
- **阿里云HTTPDNS集成**: 基于阿里云HTTPDNS服务的域名解析
- **智能代理服务器**: 本地代理服务器，支持HTTP/1.1和HTTP/2协议
- **多客户端支持**: 完美支持Dio、HttpClient、WebSocket等主流HTTP客户端
- **端口管理**: 智能端口分配和冲突处理，无需手动管理

#### 🔧 技术特性
- **DNS解析**: 支持域名解析，内置缓存机制和系统DNS回退
- **代理转发**: 自动解析域名IP，转发HTTP/HTTPS请求到目标服务器
- **协议支持**: 支持HTTP/1.1、HTTP/2、HTTPS、WebSocket协议
- **错误处理**: 完善的错误处理和重试机制

#### 📱 平台支持
- **Android**: 支持API 21+
- **iOS**: 支持iOS 12.0+

#### 📚 文档和示例
- 完整的README文档，包含使用指南和API参考
- 详细的网络请求流程图和工作原理说明
- 完整的示例项目，包含基础测试和高级功能测试
- 常见问题解答和最佳实践指南

#### 🎯 使用场景
- 移动应用网络优化
- 多端统一网络处理
- 网络调试和测试
